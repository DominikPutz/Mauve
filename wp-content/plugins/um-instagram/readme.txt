== Description ==

With our Instagram extension, users can connect to their instagram account and display the latest 18 photos on their profile.

= Key Features: =

* Users can connect & disconnect to their Instagram account
* Displays the latest 18 photos from Instagram
* Photos are displayed in a 3×2 grid on profile
* Photo grid has left and right control buttons
* Photos open in modal when clicked on
* Adds a button below grid which takes users to the Instagram account of that user

== Installation ==

1. Activate the plugin
2. That's it. Go to Ultimate Member > Settings > Extensions > Instagram to customize plugin options
3. For more details, please visit the official [Documentation](http://docs.ultimatemember.com/article/191-instagram-setup) page.
